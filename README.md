# Tedu_note
# 云笔记项目
用户可在该系统中记录自己的日常学习/旅游笔记，用户的数据将被安全的存储在云笔记平台；用户与用户之间数据为隔离存储（用户只有在登陆后才能使用相关笔记功能，且只能查阅自己的笔记内容）
- 用户模式
	- 注册 - 成为平台用户
	- 登录 - 校验用户身份
	- 退出登录 - 退出登陆状态
- 笔记模块
	- 查看笔记列表 - 查
	- 创建新笔记 - 增
	- 修改笔记 - 改
	- 删除笔记 - 删
